Instructions for use.

track_b allows you to track your packages between Benin and China. 
It is based on the trackingmore API.
So you need to create an account on trackingmore.com, in order to get your API key

To use track_b, you must first install and activate the WooCommerce plugin

Then download the track_b plugins.
Once the download is complete, go to the admin part of WordPress,
Selected extensions => add => Upload an extension

Then choose the zipped track_b file to download then you just have to click and activate the plugin.

Once activated, create a page, give the beach the name you want.
For the content of the page, you must put the following shotcode: [track_code]

You must then click on Track in the admin part and enter, that API key
in order to allow your customers to send you tracking numbers.
